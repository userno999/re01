《Web应用基础》课程结业报告

一、做的什么

本次结业项目一个介绍山西省的网页，其中分别讲述了山西省的历史沿革、行政区划、地理环境、自然资源等方面的内容。网页当中运用了最基础一些HTML标签来构建网页基础和一些比较基础的CSS试图让页面更加优美，但是我高估了自己的审美和天赋，然后就做出来个山西省介绍。

二、开发过程

没什么特别的，就先学习，后做网页，遇到不会的上网查。

三、遇到的问题

我觉的，我个人遇到的最大的问题是没有艺术细胞，外加审美，在我看来网页要么就极其优美，要么就平平淡淡，当然，比我写的好。

四、总结

HTML：学习了 HTML 的基本结构和标签，了解了如何创建和组织网页的内容。掌握了常用标签的用法，如标题、段落、列表、表格等，能够创建简单的静态网页。

CSS：学习了 CSS 的基本语法和样式规则，掌握了如何为网页添加样式和布局。了解了选择器、盒模型、浮动和定位等概念，能够创建具有吸引力的网页设计。

JavaScript：学习了 JavaScript 的基本语法和核心概念。了解了变量、数据类型、操作符、条件语句和循环语句等基本概念。掌握了函数、对象和数组等高级概念，能够编写简单的交互式网页。

通过这门课程，我不仅了解了 Web 开发的基本知识，还获得了实际动手的机会。通过完成课程项目和练习，能够独立地创建简单的静态网页，并为其添加样式和交互性。 在未来，我计划进一步学习和实践，提升我的 Web 开发技能。我将深入学习 JavaScript 和框架，如 React 或 Vue.js，以便能够构建更复杂和交互性更强的网页和应用程序。学的平平淡淡，写的平平淡淡，当务之急可能是提升审美。 总的来说，Web 应用基础课程为我打下了坚实的基础，使我能够理解和应用 Web 开发的基本概念和技术。我相信这门课程对我未来的学习和职业发展将有很大的帮助。
